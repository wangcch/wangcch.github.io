CSS 清除float浮动

使用浮动的情况有很多，比如页面布局(float:left float:right)、取消块元素的独占一行等等。浮动对页面的主要影响是，当父盒子没有设置高，子盒子在父盒子中进行浮动。由于父盒子的高度为0，下面的元素会自动补位。这时候需要清除浮动影响。还有一点是今天遇到的所以准备整理一下，就是有些响应式界面并不是清除浮动，而是对浮动的去除。

HTTP与TCP的区别

TCP是传输层，http是应用层。http是要基于TCP连接基础上的，TCP就是单纯建立连接，不涉及任何我们需要请求的实际数据，简单的传输。http是用来收发数据，即实际应用。

HTML 移动前端viewport

在开发响应式网站时Wangcch’s CV，并未在移动窗口进行测试。结果在服务器部署测试时手机并没有自适应窗口，只是显示网页缩放版本。后来查资料，一行代码解决…

<meta name="viewport" content="width=device-width,
 initial-scale=1.0, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0"/>
